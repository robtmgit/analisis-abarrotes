# analisis-abarrotes
# Análisis de Ventas – Tienda de Abarrotes

Análisis exploratorio de datos de ventas usando Python y pandas.

## Herramientas
- Python (pandas, matplotlib)
- Google Colab
- Excel (exportación de resultados)

## Qué se analizó
- Ingresos totales por categoría de producto
- Productos más vendidos por cantidad
- Tendencia de ventas por mes

## Hallazgos principales
- Granos es la categoría con mayor ingreso (~$24k)
- Bebidas ocupa el segundo lugar (~$20k)
- Enero fue el mes con más ventas

## Archivos
- `analisis_ventas.ipynb` — notebook con el análisis completo
- `resumen_ventas.xlsx` — resumen exportado con dos hojas (por categoría y por mes)
